# 部署教程 - 本地运行

---

## 📖 教程概述

本教程将指导你如何在本地电脑运行 Ozon rFBS 利润计算器，实现：

- ✅ 完全离线使用
- ✅ 数据绝对私密
- ✅ 无需网络连接
- ✅ 完全控制环境

## 📋 准备工作

### 硬件要求

- **操作系统**：Windows 10+、macOS 10.15+、Linux
- **内存**：至少 4GB RAM
- **硬盘空间**：至少 500MB 可用空间
- **网络**：首次安装需要网络连接（后续可离线）

## 🔧 安装步骤

### 步骤 1：安装 Node.js

Node.js 是运行本项目的必需环境。

#### Windows 安装

1. **下载 Node.js**

   - 访问官网：https://nodejs.org
   - 选择 **LTS（长期支持版）** 下载
   - 当前推荐版本：20.x 或更高
   - 文件名类似：`node-v20.11.0-x64.msi`

2. **运行安装程序**

   - 双击下载的 `.msi` 文件
   - 点击 "Next" 继续
   - 勾选 "I accept the terms in the License Agreement"
   - 点击 "Next"

3. **选择安装路径**

   - 默认路径：`C:\Program Files\nodejs\`
   - 建议使用默认路径
   - 点击 "Next"

4. **选择安装组件**

   - 保持默认选择
   - **确保勾选 "npm package manager"**
   - 点击 "Next"

5. **开始安装**

   - 点击 "Install"
   - 等待安装完成
   - 点击 "Finish"

6. **验证安装**

   - 按 `Win + R`，输入 `cmd`，按 Enter
   - 在命令行输入：
     ```bash
     node -v
     ```
   - 应显示版本号，如：`v20.11.0`
   
   - 再输入：
     ```bash
     npm -v
     ```
   - 应显示 npm 版本号，如：`10.2.4`

---

#### macOS 安装

1. **下载 Node.js**

   - 访问官网：https://nodejs.org
   - 选择 **LTS（长期支持版）** 下载
   - 文件名类似：`node-v20.11.0.pkg`

2. **运行安装程序**

   - 双击下载的 `.pkg` 文件
   - 按照提示点击 "Continue"
   - 同意许可协议
   - 选择安装位置（默认即可）
   - 点击 "Install"
   - 输入管理员密码确认

3. **验证安装**

   - 打开"终端"应用（Applications → Utilities → Terminal）
   - 输入：
     ```bash
     node -v
     ```
   - 应显示版本号

---

#### Linux 安装（Ubuntu/Debian）

```bash
# 更新包管理器
sudo apt update

# 安装 Node.js
sudo apt install nodejs npm

# 验证安装
node -v
npm -v
```

### 步骤 2：获取项目代码

#### 方式一：下载 ZIP 压缩包（推荐新手）

1. **获取项目源码压缩包**

2. **解压到本地目录**
   - Windows: 右键 → 解压到当前文件夹
   - Mac: 双击压缩包自动解压
   - 推荐路径：
     - Windows: `D:\ozon-calculator`
     - Mac: `/Users/你的用户名/ozon-calculator`

---

#### 方式二：使用 Git 克隆

**前提**：已安装 Git（https://git-scm.com）

1. **打开命令行工具**
   - Windows: 按 `Win + R`，输入 `cmd`
   - Mac: 打开"终端"应用

2. **进入目标目录**
   ```bash
   # Windows
   cd D:\
   
   # Mac/Linux
   cd ~
   ```

3. **克隆项目**
   ```bash
   git clone [项目Git地址]
   ```

4. **进入项目目录**
   ```bash
   cd ozon-calculator
   ```

---

### 步骤 3：安装项目依赖

依赖是项目运行所需的第三方库。

1. **打开命令行工具**

   - Windows: 在项目文件夹中，按 `Shift + 右键`，选择"在此处打开 PowerShell 窗口"
   - Mac: 打开"终端"，使用 `cd` 命令进入项目目录

2. **进入项目目录**（如果还没进入）
   ```bash
   cd ozon-calculator
   ```

3. **安装依赖**
   ```bash
   npm install
   ```

   **等待过程**：
   - 会显示安装进度
   - 可能需要几分钟
   - 看到进度条和包名称

   **成功标志**：
   ```
   added 320 packages in 45s
   ```

4. **验证安装**
   
   检查是否生成了 `node_modules` 文件夹：
   - 该文件夹包含所有依赖库
   - 文件夹很大（约 200-300MB）是正常的

## 🚀 运行项目

### 方式零：使用一键启动脚本（最简单，推荐新手）

**特点**：
- ✅ 零命令行操作
- ✅ 自动环境检测
- ✅ 自动安装依赖
- ✅ 自动启动服务
- ✅ 自动打开浏览器

#### Windows 用户

1. **双击启动**
   - 找到项目文件夹中的 `start-local.bat`
   - 双击运行即可

2. **脚本会自动执行以下操作**：
   ```
   ✓ 检查 Node.js 环境
   ✓ 检查 npm 环境
   ✓ 显示版本信息
   ✓ 自动安装依赖（首次运行）
   ✓ 启动开发服务器
   ✓ 自动打开浏览器
   ```

3. **停止服务**
   - 在命令行窗口按 `Ctrl + C`
   - 或双击运行 `stop-local.bat` 强制停止

#### macOS/Linux 用户

1. **首次使用，添加执行权限**
   ```bash
   chmod +x start-local.sh
   ```

2. **启动服务**
   ```bash
   ./start-local.sh
   ```

3. **脚本会自动执行以下操作**：
   ```
   ✓ 检查 Node.js 环境
   ✓ 检查 npm 环境
   ✓ 显示版本信息（彩色输出）
   ✓ 自动安装依赖（首次运行）
   ✓ 启动开发服务器
   ✓ 自动打开浏览器
   ```

4. **停止服务**
   - 在终端按 `Ctrl + C`

## 🛠️ 常见问题

### Q1：`npm install` 报错怎么办？

**A**：尝试以下方法：

1. **清除 npm 缓存**
   ```bash
   npm cache clean --force
   ```

2. **删除 node_modules 重新安装**
   ```bash
   # Windows
   rmdir /s node_modules
   del package-lock.json
   
   # Mac/Linux
   rm -rf node_modules
   rm package-lock.json
   
   # 重新安装
   npm install
   ```

3. **使用国内镜像**（如果网络慢）
   ```bash
   npm install --registry=https://registry.npmmirror.com
   ```

---

### Q2：启动后页面空白？

**A**：检查以下项：

1. **查看命令行错误**
   - 是否有报错信息
   - 根据错误提示修复

2. **查看浏览器控制台**
   - 按 F12 打开开发者工具
   - 查看 Console 标签是否有错误

3. **清除浏览器缓存**
   - 按 `Ctrl + Shift + Delete`
   - 清除缓存和 Cookie

4. **使用无痕模式测试**
   - 排除插件干扰

---

### Q3：端口被占用怎么办？

**A**：默认端口 3000 被占用，Next.js 会自动使用下一个端口：

```
⚠ Port 3000 is in use, using available port 3001 instead.
```

浏览器访问显示的端口即可（如 `http://localhost:3001`）。

**手动指定端口**：
```bash
# Windows PowerShell
$env:PORT=3005; npm run dev

# Mac/Linux
PORT=3005 npm run dev
```

---

### Q4：如何停止后台运行的服务？

**A**：

**Windows**：
```bash
# 查找占用 3000 端口的进程
netstat -ano | findstr :3000

# 结束进程（PID 从上一步获取）
taskkill /PID <进程ID> /F
```

**Mac/Linux**：
```bash
# 查找并结束进程
lsof -ti:3000 | xargs kill -9
```

## 🚫 防火墙设置

### Windows 防火墙

如果启动后无法访问，可能需要允许 Node.js 通过防火墙：

1. **打开防火墙设置**
   - 控制面板 → 系统和安全 → Windows Defender 防火墙
   - 点击"允许应用通过防火墙"

2. **添加 Node.js**
   - 点击"更改设置"
   - 点击"允许其他应用"
   - 浏览找到 `node.exe`（通常在 `C:\Program Files\nodejs\`）
   - 勾选"专用"和"公用"
   - 点击"确定"

---

### Mac 防火墙

1. **系统偏好设置** → **安全性与隐私** → **防火墙**
2. 点击"防火墙选项"
3. 允许 Node.js 接受传入连接

## 📝 快速启动脚本（可选）

### Windows 批处理脚本

创建 `start.bat` 文件：

```batch
@echo off
echo 启动 Ozon rFBS 利润计算器...
cd /d %~dp0
start http://localhost:3000
npm start
pause
```

双击 `start.bat` 即可启动。

---

### Mac Shell 脚本

创建 `start.sh` 文件：

```bash
#!/bin/bash
echo "启动 Ozon rFBS 利润计算器..."
cd "$(dirname "$0")"
open http://localhost:3000
npm start
```

运行：
```bash
chmod +x start.sh
./start.sh
```
